# Letter_Move_Site
３種類の言葉があなたの画面を彩ります。

JavaScriptとCSSアニメーションを使って、テキストをクリックすると背景がユニークな動きをするWebアプリケーションです。

## 使い方
1. このリポジトリをダウンロードまたはクローンします。
2. `index.html` をブラウザで開くだけで動作を確認できます。

## ファイル構成
* `index.html` : 画面の骨組みと各ファイルの読み込み
* `style.css` : テキストのデザインと移動アニメーションの定義
* `script.js` : クリックイベントの検知とクラスの切り替え制御

文字を押した時のエフェクト効果時間や、背景、文字の動きがどんなものだとワクワクするかに力を入れました。
